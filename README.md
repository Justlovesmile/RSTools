# Introduction

Tools for remote sensing images (RSIs) processing and visualization.

遥感图像与标签处理工具箱，主要用于存储个人常用切图、格式转换、可视化等代码。

# Installation

```bash
python setup.py build develop # or `pip install -v -e .`
```

# Usage

```python
from rstools.func import *
from rstools.utils import *
# ......
```
